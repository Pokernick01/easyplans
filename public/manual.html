1. Overview 1. Descripción General

EasyPlans is a free, open-source architectural floor plan drawing application built for students and professionals. It runs entirely in your web browser with no installation required.

Key features include:

  • Draw walls, doors, windows, rooms, stairs, and furniture
  • Two active view modes: Plan and 3D Isometric
  • Export to PNG, transparent PNG, or PDF
  • Multi-floor support with independent element sets
  • Bilingual interface (English / Spanish)
  • Neufert-style architectural drawings
  • Auto-save to browser storage
  • Custom .easyplan project files

All measurements are in meters. The coordinate system uses X (horizontal) and Y (vertical) with the origin at the center of the canvas.

EasyPlans es una aplicación gratuita y de código abierto para dibujar planos arquitectónicos, creada para estudiantes y profesionales. Funciona completamente en tu navegador web sin necesidad de instalación.

Características principales:

  • Dibuja muros, puertas, ventanas, cuartos, escaleras y muebles
  • Dos modos de vista activos: Planta e Isométrico 3D
  • Exportar a PNG, PNG transparente o PDF
  • Soporte multi-piso con elementos independientes
  • Interfaz bilingüe (Inglés / Español)
  • Dibujos arquitectónicos estilo Neufert
  • Autoguardado en almacenamiento del navegador
  • Archivos de proyecto personalizados .easyplan

Todas las medidas son en metros. El sistema de coordenadas usa X (horizontal) e Y (vertical) con el origen en el centro del lienzo.

2. Interface Layout 2. Distribución de la Interfaz

Top Bar

Contains the logo, language toggle (EN/ES), view mode tabs (Plan and 3D), scale selector, front-direction selector (North/East/South/West), grid/snap/dimension toggles, undo/redo, new project, clear canvas, export, save/load buttons, help button, and a prominent Apoyame / Support me button.

Left Sidebar

Tool palette with all drawing tools. Click a tool or use its keyboard shortcut to activate it. The active tool is highlighted in green.

Canvas Area

The main drawing area. Pan by scrolling (mouse wheel or trackpad), right-click dragging, middle-click dragging, or holding Space + left-click drag. Ctrl+scroll to zoom. Shift+scroll for horizontal pan. Arrow keys pan when nothing is selected. Scrollbars appear when zoomed in. All drawing happens here.

Right Sidebar

Two tabs:

  • Properties — Edit selected element properties (dimensions, colors, patterns)
  • Library — Browse and select furniture/stamps for placement

Bottom Bar

Shows cursor coordinates (X, Y in meters), floor tabs for multi-floor projects, active tool name and hint, zoom controls, and zoom-to-fit button.

Language Toggle

Click EN or ES in the top-left to switch between English and Spanish. All labels, tooltips, and property names update instantly.

Barra Superior

Contiene el logo, selector de idioma (EN/ES), pestañas de modo de vista (Planta y 3D), selector de escala, selector de dirección de frente (Norte/Este/Sur/Oeste), botones de cuadrícula/ajuste/cotas, deshacer/rehacer, nuevo proyecto, limpiar lienzo, exportar, guardar/cargar, botón de ayuda y un botón Apóyame / Support me más visible.

Barra Lateral Izquierda

Paleta de herramientas con todas las herramientas de dibujo. Haz clic en una herramienta o usa su atajo de teclado para activarla. La herramienta activa se resalta en verde.

Área del Lienzo

El área principal de dibujo. Desplaza con la rueda del ratón (scroll), arrastrando con clic derecho, clic medio, o manteniendo Espacio + clic izquierdo. Ctrl+scroll para hacer zoom. Shift+scroll para desplazar horizontalmente. Las flechas desplazan cuando no hay nada seleccionado. Las barras de desplazamiento aparecen al hacer zoom. Todo el dibujo ocurre aquí.

Barra Lateral Derecha

Dos pestañas:

  • Propiedades — Edita las propiedades del elemento seleccionado (dimensiones, colores, patrones)
  • Biblioteca — Explora y selecciona muebles/sellos para colocación

Barra Inferior

Muestra las coordenadas del cursor (X, Y en metros), pestañas de pisos para proyectos multi-piso, nombre de herramienta activa y pista, controles de zoom y botón de ajustar a vista.

Selector de Idioma

Haz clic en EN o ES en la esquina superior izquierda para cambiar entre inglés y español. Todas las etiquetas, tooltips y nombres de propiedades se actualizan instantáneamente.

5. Walls 5. Muros

Walls are the foundation of any floor plan. They connect two endpoints and have configurable thickness and height.

Drawing Walls

  1. Press W to activate the Wall tool
  2. Click to place the start point (snaps to grid and existing endpoints)
  3. Move the cursor to see a preview line
  4. Click again to place the end point and create the wall
  5. Chain mode: Hold Shift on the second click to start a new wall from the endpoint
  6. Press Esc to cancel

Los muros son la base de cualquier plano. Conectan dos puntos y tienen espesor y altura configurables.

Dibujar Muros

  1. Presiona W para activar la herramienta de Muro
  2. Clic para colocar el punto de inicio (se ajusta a la cuadrícula y puntos existentes)
  3. Mueve el cursor para ver una vista previa de la línea
  4. Clic de nuevo para colocar el punto final y crear el muro
  5. Modo cadena: Mantén Shift en el segundo clic para iniciar un nuevo muro desde ese punto
  6. Presiona Esc para cancelar

8. Rooms & Patterns 8. Cuartos y Patrones

Rooms auto-detect enclosed wall areas. Draw a complete wall perimeter first, then use the Room tool to click inside.

Creating a Room

  1. Draw walls forming a closed polygon (at least 3 walls)
  2. Press R to activate the Room tool
  3. Click inside the enclosed area
  4. The room auto-detects boundaries and fills with a label
  5. Edit name, color, and pattern in the Properties panel

Crear un Cuarto

  1. Dibuja muros formando un polígono cerrado (al menos 3 muros)
  2. Presiona R para activar la herramienta de Cuarto
  3. Haz clic dentro del área cerrada

12. Dimensions & Text 12. Cotas y Texto

Dimensions

Dimension lines show distances between two points with automatic measurement labels.

  1. Press M to activate the Dimension tool
  2. Click the start point
  3. Click the end point
  4. Adjust the offset distance in Properties to move the line away from the wall

Text Labels

Free-form text labels for annotations, room names, or notes.

  1. Press T to activate the Text tool
  2. Click anywhere on the canvas
  3. Edit the text content, font size, color, and rotation in the Properties panel
Tip: Toggle dimension visibility with the Dimensions button in the top bar. This hides/shows all dimension lines on the canvas.

Cotas

Las líneas de cota muestran distancias entre dos puntos con etiquetas de medición automáticas.

  1. Presiona M para activar la herramienta de Cota
  2. Haz clic en el punto de inicio
  3. Haz clic en el punto final
  4. Ajusta la distancia de desplazamiento en Propiedades para mover la línea lejos del muro

Etiquetas de Texto

Etiquetas de texto libre para anotaciones, nombres de cuartos o notas.

  1. Presiona T para activar la herramienta de Texto
  2. Haz clic en cualquier lugar del lienzo
  3. Edita el contenido del texto, tamaño de fuente, color y rotación en el panel de Propiedades
Consejo: Activa/desactiva la visibilidad de cotas con el botón Cotas en la barra superior. Esto oculta/muestra todas las líneas de cota en el lienzo.

13. View Modes 13. Modos de Vista

Plan View (Planta)

Default orthographic top-down view. Shows all walls, openings, furniture, rooms, dimensions, and annotations. This is where most design work happens.

A compass marker is shown on canvas/export with true North and the configured Front direction.

Isometric View (3D)

Pseudo-3D axonometric projection showing the building in three dimensions. Walls display their full height, doors and windows show their openings. Rotation and elevation can be adjusted.

3D now includes improved shading, cleaner depth layering, and better orbit controls for mouse wheel and touch gestures.

A compass marker is also shown in 3D/export to keep orientation clear (North + Front).

Note: Section and Facade tabs are currently disabled in the active UI flow to prioritize stability and quality in Plan + 3D.

Vista de Planta

Vista ortográfica superior predeterminada. Muestra todos los muros, aberturas, muebles, cuartos, cotas y anotaciones. Aquí es donde ocurre la mayoría del trabajo de diseño.

Se muestra un marcador de orientación en lienzo/exportación con Norte real y la dirección de frente configurada.

Vista Isométrica (3D)

Proyección axonométrica pseudo-3D mostrando el edificio en tres dimensiones. Los muros muestran su altura completa, las puertas y ventanas muestran sus aberturas. Se puede ajustar rotación y elevación.

El 3D ahora incluye mejor sombreado, mejor orden de profundidad y mejores controles de órbita para rueda del mouse y gestos táctiles.

También se muestra un marcador de orientación en 3D/exportación para mantener clara la orientación (Norte + Frente).

18. Tips & Workflows 18. Consejos y Flujos de Trabajo

Efficient Wall Drawing

Chain mode: When drawing walls, hold Shift on the second click to immediately start a new wall from that endpoint. This lets you quickly draw entire room perimeters without restarting.

Quick Room Creation

  1. Draw 4 walls forming a rectangle (use chain mode)
  2. Switch to Room tool (R)
  3. Click inside the rectangle
  4. Name and style the room in Properties

Furnishing a Room

  1. Switch to Furniture tool (F)
  2. Open the Library tab and browse categories
  3. Click an item, move to position, press R to rotate if needed
  4. Click to place, repeat for more items

Professional Exports

Best practice: Set your scale first (e.g., 1:100), enable the title block, fill in project name and author, then export as PDF for print-ready architectural drawings.

Multi-Floor Workflow

  1. Complete the ground floor plan
  2. Click + to add a new floor, or duplicate the ground floor
  3. Switch between floors using the tabs
  4. Each floor maintains independent elements
  5. 3D view can show the stacked massing across floors

Using Architectural Lines

Site plans: Use Boundary (Colindancia) lines for property boundaries, Lot Limit lines for lot limits, and Axis (Eje) lines for structural axes. These follow Latin American architectural conventions.
Remember: Save your project regularly with Ctrl+S. While auto-save helps, browser storage can be cleared.

Dibujo Eficiente de Muros

Modo cadena: Al dibujar muros, mantén Shift en el segundo clic para iniciar inmediatamente un nuevo muro desde ese punto. Esto te permite dibujar rápidamente perímetros completos sin reiniciar.

Creación Rápida de Cuartos

  1. Dibuja 4 muros formando un rectángulo (usa modo cadena)
  2. Cambia a la herramienta Cuarto (R)
  3. Haz clic dentro del rectángulo
  4. Nombra y dale estilo al cuarto en Propiedades

Amueblar un Cuarto

  1. Cambia a la herramienta de Muebles (F)
  2. Abre la pestaña Biblioteca y explora las categorías
  3. Haz clic en un elemento, mueve a posición, presiona R para rotar si es necesario
  4. Clic para colocar, repite para más elementos

Exportaciones Profesionales

Mejor práctica: Establece tu escala primero (ej., 1:100), activa el cuadro de título, llena el nombre del proyecto y autor, luego exporta como PDF para dibujos arquitectónicos listos para imprimir.

Flujo de Trabajo Multi-Piso

  1. Completa el plano de planta baja
  2. Clic en + para agregar un nuevo piso, o duplica la planta baja
  3. Cambia entre pisos usando las pestañas
  4. Cada piso mantiene elementos independientes
  5. La vista 3D puede mostrar la volumetría apilada entre pisos

Usando Líneas Arquitectónicas

Planos de sitio: Usa líneas de Colindancia para límites de propiedad, Límite de Lote para límites de lote, y Eje para ejes estructurales. Estas siguen las convenciones arquitectónicas latinoamericanas.
Recuerda: Guarda tu proyecto regularmente con Ctrl+S. Aunque el autoguardado ayuda, el almacenamiento del navegador puede limpiarse.
